gtasks-mcp

A private Model Context Protocol (MCP) server for Google Tasks

Private Use Only

About

gtasks-mcp is a self-hosted MCP server that connects AI assistants (such as Claude) to a personal Google Tasks account. It allows an authorised user to read, create, update, and delete tasks through natural-language interactions with an AI assistant.

This application is privately operated and is not available to the general public. Access is restricted to authorised Google accounts.

What This App Does

  • Reads your Google Tasks lists and individual tasks
  • Creates new tasks and updates existing ones
  • Marks tasks as complete or deletes them
  • Exposes this functionality to an AI assistant via the MCP protocol

Google API Scope Used

This application requests the following OAuth 2.0 scope from Google:

  • https://www.googleapis.com/auth/tasks — Read and write access to Google Tasks. This is the minimum scope required to manage tasks on your behalf.

Data & Privacy

No personal data is transmitted to or stored on any third-party server. OAuth credentials are stored locally on the machine running this server and are used solely to authenticate requests to the Google Tasks API on your behalf.
